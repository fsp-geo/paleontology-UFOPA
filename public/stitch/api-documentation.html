Developer & Researcher Integration

The Strata Archive API provides direct access to over 150 years of paleontological records, stratigraphic columns, and high-resolution micro-fossil scans.

Authentication

To access the Petrobras Paleontology Division's archival data, all requests must be signed with a Bearer Token. Researchers can generate long-lived keys within the Access Portal.

HTTP
GET /v2/stratigraphy/search HTTP/1.1
Host: api.strataarchive.org
Authorization: Bearer YOUR_API_KEY
Accept: application/json

Stratigraphic Querying

The core endpoint for searching historical geological layers. You can filter by basin, period, or fossil presence.

GET /stratigraphy/basins/{basin_id}

Parameters

basin_id String • Required

The unique identifier for the sedimentary basin (e.g., 'campos_01').

Response Schema

{
  "basin_name": "Campos Basin",
  "period": "Late Cretaceous",
  "fossil_count": 1420,
  "depth_meters": 2500.5
}
